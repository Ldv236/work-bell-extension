# Privacy Policy for Work Bell: Breaks and Exercises

Last updated: March 8, 2026

Work Bell: Breaks and Exercises does not collect, transmit, sell, or share personal data with third parties.

## What the extension stores

The extension stores the following data in Chrome extension storage:

- user settings, such as active hours, reminder interval, repeat interval, signal type, voice phrase settings, volume, quick sound mute preference, queue mode, and exercise list;
- local runtime state required for reminder scheduling, such as the next reminder time, the active reminder state, pause state, deferred exercise, daily queue, today's completed exercises, and completion history by day.

## Where data is stored

All data is stored locally in the user's browser using Chrome extension storage APIs.

The extension does not use an external database, backend server, analytics service, advertising service, or third-party tracker.

## What the extension does not collect

The extension does not collect or access:

- browsing history;
- web page content;
- search queries;
- account credentials;
- contacts;
- health records;
- financial information;
- location data;
- personal communications.

## How the extension uses permissions

The extension uses only the following Chrome permissions:

- `alarms` to schedule reminders;
- `storage` to save settings and local reminder state;
- `notifications` to display reminder notifications;
- `offscreen` to play voice or sound signals.

## Data sharing

No user data is sold, transferred, or shared with third parties.

## Remote code

The extension does not download, execute, or use remote code.

## Changes to this policy

If the extension's data handling changes in the future, this privacy policy will be updated before those changes are released.

## Contact

For questions about this privacy policy, use the support contact listed on the Chrome Web Store page for this extension.
