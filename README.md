# Work Bell - упражнения

Work Bell - это расширение Chrome/Chromium Manifest V3 для регулярных напоминаний о коротких упражнениях в течение рабочего дня. Расширение показывает настойчивое напоминание, воспроизводит голосовой или звуковой сигнал и повторяет его, пока пользователь не отметит упражнение как выполненное или пропущенное.

## Что делает расширение

- Работает ежедневно только в заданном временном окне, по умолчанию с 09:00 до 22:00.
- Создает напоминания с выбранным упражнением из пользовательского списка.
- Показывает системное уведомление Chrome с кнопками "Сделано" и "Пропущу".
- Открывает отдельное popup-окно напоминания поверх обычного интерфейса расширения.
- Ставит badge `!` на иконку расширения, пока есть активное напоминание.
- Повторяет активный сигнал через заданный интервал, пока напоминание не будет закрыто.
- Позволяет выполнить, пропустить или отложить активное упражнение.
- Ведет дневную статистику выполненных упражнений и сохраняет историю по дням.
- Позволяет настраивать расписание, список упражнений, порядок очереди, звук, голосовую фразу и громкость.
- Позволяет быстро выключить или включить звук из обычного popup без отключения расширения и без изменения сохраненной громкости.
- Позволяет включить режим "Не беспокоить" на 30, 60 или 90 минут без выключения расширения.
- Поддерживает отдельные вечерние напоминания после окончания дневного периода: они настойчиво повторяют заданный текст, пока пользователь не нажмет "Пошел спать".
- Поддерживает экспорт и импорт настроек, очереди и истории в JSON.
- Позволяет создать тестовое напоминание с настоящим уведомлением и отдельным окном.

## Основные сценарии

### Обычный popup

Открывается по иконке расширения. Показывает текущий статус, следующее время сигнала, активное упражнение, кнопки действия и краткую статистику за текущий день.

Кнопка "Сделано" закрывает активное напоминание и учитывает упражнение в дневной статистике. Кнопка "Пропущу" закрывает активное напоминание без учета.

Кнопка "Отложить" закрывает активное напоминание без учета и возвращает текущее упражнение в начало очереди. Следующее напоминание снова будет про это же упражнение, а порядок остальных упражнений не сдвинется.

Чекбокс "Звук" управляет быстрым mute-режимом и показывает состояние цветом: включено зеленым, выключено красным. В mute-режиме расписание, уведомления, окно напоминания и статистика продолжают работать, но голосовой и звуковой сигнал не воспроизводятся. При включении звука обратно используется прежняя громкость из настроек.

Блок "Не беспокоить" ставит паузу на новые напоминания на 30 минут, 1 час или 1.5 часа. Если включить паузу при активном напоминании, текущее упражнение возвращается в очередь и снова будет предложено после паузы. Во время паузы popup показывает время окончания паузы и следующее упражнение, а кнопка "Снять паузу" возвращает расписание в работу.

Кнопка "История" показывает выполненные упражнения за сегодня с временем выполнения, а также сохраненные итоги прошлых дней.

### Окно напоминания

При наступлении времени сигнала `background.js` открывает `popup.html?mode=reminder` в отдельном окне. В этом режиме интерфейс крупнее, скрывает настройки и историю дня, оставляя только упражнение и действия "Сделано" / "Отложить" / "Пропущу".

Для вечернего напоминания это же окно становится минимальным: показывает только вечерний текст и одну кнопку "Пошел спать". Быстрый звук, история, настройки и пауза в этом окне не показываются.

### Страница настроек

`options.html` позволяет настроить:

- рабочее временное окно;
- режим расчета интервала;
- основной интервал между новыми упражнениями;
- интервал повторного сигнала для активного напоминания;
- тип сигнала: голос или короткий звук;
- начало и конец голосовой фразы;
- громкость;
- вечерние напоминания: включение, конец вечернего периода, интервал повтора и текст;
- порядок очереди упражнений;
- список упражнений, по одному на строку;
- ручное обновление очереди;
- тестовое воспроизведение сигнала;
- тестовое напоминание с уведомлением и отдельным окном;
- экспорт и импорт данных в JSON.

## Режимы расписания

Расширение поддерживает два режима.

`after_confirmation` - следующий сигнал считается от момента подтверждения или пропуска предыдущего напоминания. Это режим по умолчанию.

`fixed_slots` - сигналы привязаны к фиксированной сетке времени от начала рабочего окна. Если слот был пропущен, расширение не пытается догонять старые слоты.

Если время "с" не раньше времени "по", расписание считается некорректным: новые напоминания не создаются, а интерфейс показывает ошибку настроек.

Если вечерние напоминания включены, дневное "Время по" одновременно считается началом вечернего периода. Вечернее "время по" должно быть позже дневного окончания. Внутри вечернего периода создается отдельное напоминание с пользовательским текстом; оно повторяется с вечерним интервалом, пока не нажата кнопка "Пошел спать". После подтверждения вечерние напоминания до следующего дня не создаются.

## Очередь упражнений

В начале дня расширение формирует дневную очередь из списка упражнений. Очередь может быть случайной или идти в порядке списка. Внутри одной очереди упражнения не повторяются; когда очередь заканчивается, создается новая очередь по выбранному правилу.

Состояние очереди сбрасывается при смене календарного дня. Кнопка "Обновить очередь" на странице настроек пересоздает очередь вручную.

При откладывании упражнение возвращается в начало очереди, а в popup появляется пометка, что оно повторится следующим.

Текущий список упражнений по умолчанию:

- приседания;
- отжимания от стены или стола;
- наклоны вперед;
- наклоны в стороны;
- круговые движения плечами;
- махи руками;
- подъемы на носки;
- ходьба на месте.

В коде есть миграция старых списков упражнений и старых фраз напоминания к текущим значениям по умолчанию.

## Звук и голос

Для воспроизведения расширение использует offscreen-документ `offscreen.html` и скрипт `offscreen.js`.

В режиме `voice` используется Web Speech API. Расширение пытается выбрать русскоязычный голос, затем англоязычный, затем первый доступный голос. Если синтез речи недоступен или завершается ошибкой, используется fallback-звук `sounds/bell.wav`.

В режиме `beep` всегда используется короткий звук из `sounds/bell.wav`.

Первый голосовой сигнал произносит полную фразу из начала, названия упражнения и окончания. Повтор активного напоминания произносит только название упражнения.

Быстрый mute-флаг `soundMuted` подавляет воспроизведение сигналов и останавливает текущую речь или звук при включении mute. Флаг не меняет значение `volume`.

## История, пауза и тесты

Каждое действие "Сделано" записывает упражнение в `completedToday` и в накопительную историю `historyByDay`. Popup показывает подробности текущего дня и прошлых дней с временем каждого выполненного упражнения.

Режим "Не беспокоить" хранится как `pausedUntil` в локальном состоянии. Пока пауза активна, новые напоминания не создаются, уведомления и окно активного напоминания очищаются, а после окончания паузы расписание продолжает работу.

Тестовое напоминание создается через настройки. Оно показывает настоящее уведомление, открывает окно напоминания, ставит badge и воспроизводит сигнал, но не сдвигает очередь и не записывает выполнение в историю.

Экспорт JSON включает настройки и локальное состояние: очередь, историю, текущую дневную статистику, отложенное упражнение и активную паузу. Импорт JSON восстанавливает эти данные и пересчитывает расписание.

## Как устроен проект

Проект не использует сборщик, package manager или внешние зависимости. Это набор статических файлов расширения.

- `manifest.json` - Manifest V3, разрешения, service worker, popup, options page и иконки.
- `background.js` - основная логика расписания, очереди упражнений, состояния, alarm-таймеров, уведомлений, popup-окна и сообщений.
- `popup.html` / `popup.js` - интерфейс статуса, активного упражнения, действий и статистики дня.
- `options.html` / `options.js` - интерфейс настроек, валидация, сохранение настроек, тест сигнала и сброс очереди.
- `offscreen.html` / `offscreen.js` - воспроизведение голоса или fallback-звука через offscreen API.
- `sounds/bell.wav` - локальный звуковой сигнал.
- `icons/` - иконки расширения.
- `PRIVACY_POLICY.md` - политика приватности для публикации.

- `endTime`;
- `bedtimeEnabled`;
- `bedtimeEndTime`;
- `bedtimeIntervalMinutes`;

- `scheduleMode`;

- `reminderOutroText`;

- `nextDueAt`;

Расширение не обращается к сайтам, не использует backend, аналитику, внешние трекеры или remote code.

## Используемые Chrome API

- `chrome.alarms` - минутный тик расписания и повтор активного напоминания.
- `chrome.storage` - настройки и локальное состояние.
- `chrome.notifications` - системные уведомления с действиями.
- `chrome.offscreen` - воспроизведение звука и речи из фонового процесса.
- `chrome.windows` - отдельное окно активного напоминания.
- `chrome.action` - badge на иконке расширения.

## Установка для разработки

1. Откройте `chrome://extensions`.
2. Включите Developer mode.
3. Нажмите Load unpacked.
4. Выберите директорию проекта `work-bell-extension`.

После изменения файлов расширение нужно перезагрузить на странице `chrome://extensions`. Отдельной команды сборки нет.

## Особенности и ограничения

- Фоновый service worker проверяет состояние через alarm `work-bell-tick` раз в минуту.
- Активное напоминание восстанавливается при запуске браузера или расширения.
- Если активное напоминание осталось с предыдущего дня, оно очищается.
- Если пользователь закрыл окно напоминания, повторный alarm откроет его снова, пока напоминание активно.
- Дневная статистика учитывает только действия "Сделано".
- Пользовательский список упражнений ограничивается первыми 200 непустыми строками.
